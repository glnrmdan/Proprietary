Installasi Windows

Setelah Virtual Machine terbentuk dan ditekan tombol Start pada Menu Virtual Box. Maka Virtual Machine kita akan berjalan. Tunggulan beberapa saat sampai tampilan instalasi windows berikut ini muncul :

Pada tahapan ini kita atur dulu bahasa yang nanti akan digunakan dalam proses instalasi, zona waktu dan jenis keyboard kita. Setelah itu tekan Next. Akan muncul tampilan seperti berikut ini untuk selanjutnya kalian tekan tombol Install Now.

Jenis OS

Tampilan selanjutnya adalah memilih jenis instalasi. Disini kita gunakan STD atau Standar instalation saja kemudian tekan next untuk, setelah ditekan next akan muncul mengenai persetujuan lisensi. Tekan saja next.

Tipe Instalasi

Langkah berikutnya adalah memilih tipe instalasi. Pilih saja yang “Custom: Install Windows Only” tidak apa apa.

Setelah itu kita pilih Virtual Drive yang sudah kita buat dari tahap persiapan awal tadi. Kita pilih saja Drive 0 ini kemudian tekan tombol Next yang ada dibawah.

Proses Instalasi

Disinilah kesabaran akan diuji, proses instalasi windows seringkali memakan beberapa waktu. Jadi bersabarlah untuk menunggunya sampai selesai.

Personalize

Windows berhasil terinstall didalam komputer kita. Kita bisa memberikan nama untuk Personal Computer kita. Setelah itu tekan tombol Next.

Settings

Dibagian ini kita akan diberikan rekomendasi mengenai settingan untuk windows kita. Pilih saja langsung yang Use Express Settings.

Account

Account atau user ini nantinya digunakan disaat windows selesai booting dan masuk kedalam lock screen dari windows. Kita bisa mengisikan Username dengan nama kita dan untuk kolom password kita bisa isikan dengan password yang mudah untuk diingat. Setelah itu tekan tombol Finish.

Selesai

Selamat ! Windows anda telah terinstall dan berikut ini adalah tampilan dekstopnya :
