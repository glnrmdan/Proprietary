Tahap Persiapan

Download dan Install terlebih dahulu aplikasi virtual box kedalam komputer anda.

Untuk mengunduh virtual box anda bisa membuka websitenya dengan cara menekan disini

Setelah anda berhasil mengunduh dan menginstall virtual box maka bukalah aplikasinya.
Berikut ini adalah tampilan awal virtual box :

Membuat Virtual OS

Langkah selanjutnya adalah membuat virtual OS. Caranya adalah dengan menekan tombol new yang ada dimenu bagian atas dari Virtual Box.

Setelah anda menekan tombol new maka akan muncul box ditengah. Isikan bagian “Name” dengan jenis OS yang anda install. Disini saya menggunakan Windows 8. Berikut ini adalah tampilannya :

Mengatur Ukuran Memory

Setelah itu kita akan masuk pada tahapan mengatur ukuran Memory atau RAM yang akan digunakan oleh virtual komputer kita. Disarankan agar ukuran dari Memory kita tidak sampai memasuki zona merah, karena dikhawatirkan komputer kita akan mengalami crash. Zona maksimal untuk penggunaan wajar hanyalah sampai di tengah-tengah saja, dan jangan lebih.

Virtual Hardisk

Langkah selanjutnya adalah membuat virtual hardisk yang nantinya akan berfungsi sebagai tempat penyimpanan dari virtual komputer kita. Berikut ini adalah langkah-langkahnya :

Virtual Machine Terbentuk

Pada tahapan ini Virtual Machine kita untuk windows 8 telah selesai dibuat. Setelah itu tekanlah gambar Settings yang ada dibagian menu atas dari Virtual Box. Setelah anda tekan maka akan muncul menu baru. Lalu tekanlah bagian System yang ada dipanel sebelah kiri dan buka menu Processor yang ada didalamnya. Processor ini nantinya adalah jumlah processor yang akan digunakan oleh Virtual Machine kita. Sama seperti sebelumnya, untuk mengaturnya disarankan jangan sampai memasuki zona merah.
